about me.

Hi, my name is Saksham.


I'm current doing my undergrad for Computer Engineering at the University of Waterloo


I like writing tiny, efficient code that turns into software that's extensible, portable and beautiful. I think that if the programming world were built upon minimal viable programs it would be simpler, more connected and faster.


Software that communicates in a common language interacts with itself in a multiplicative manner, rather than additive as most software exists now. I understand these ideals are not always completely realistic when writing industry code as there's deadlines and other factors that occassionally make the delta between what I'd like to achieve and how much I can achieve not quite as small as I'd like it, but nonetheless, these are things I strive for.

In comparison to others in the industry, I'm new and, perhaps, naive, but I want take steps towards writing cleaner code that's readable and accessible by everyone.
